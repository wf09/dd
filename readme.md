## 1.确保已安装以下软件：

```css
apt-get update
apt-get install -y xz-utils openssl gawk file
```

## 2.下载及说明：

```shell
wget https://cdn.jsdelivr.net/gh/wf09/dd/InstallNET.sh
chmod a+x InstallNET.sh
```

## 3.安装Ubuntu16.04（大约要等半个小时）:

```text
bash InstallNET.sh -u 16.04 -v 64 -a
```
